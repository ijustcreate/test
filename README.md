# Felix d&d

**Every roll made the map a little more real.**

A public, responsive family D&D campaign website for Ember, Astral, Felix and guests.

## Included

- 18-page illustrated interactive storybook
- 80-entry linked campaign atlas with aliases, canon labels and source provenance
- 29-event chronicle, dialogue archive, table canon and unresolved threads
- stylized world map and relationship graph
- responsive character sheets and party HP tracking
- dice roller with modifiers, advantage/disadvantage and history
- initiative tracker and soft-combat encounter library
- draggable token battle map
- active quests, campaign clock, notes and session log
- local autosave, portable JSON export/import and reset controls
- installable/offline-capable PWA shell
- GitHub Pages deployment workflow

## Local use

No build step is required. Serve the repository with any static web server:

```bash
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Table saves

The hosted site is static. Table state is stored in the current browser using `localStorage`. Use **Save & export** to move a campaign save between desktop, iPhone and tablet browsers.
